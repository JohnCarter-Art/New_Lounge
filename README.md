# New_Lounge — Гостевая Head First (адаптивная версия)

Учебный проект из книги **«Head First HTML and CSS»** — сайт вымышленного бара/гостевой.
Включает адаптивный дизайн (десктоп, мобильный, печать).

## Файлы

- `lounge.html` — Главная страница (русский/английский)
- `lounge.css` — Стили для десктопа
- `lounge-mobile.css` — Стили для мобильных устройств
- `lounge-print.css` — Стили для печати
- `about/directions.html` — Страница с маршрутом
- `beverages/elixir.html` — Страница с напитками

## Исправленные баги

- CSS для печати и мобилки использовал `#guarantee`, а в HTML был `#quarantee` — унифицировано
- Битый фон `elixirbackground.gif` (не существовал) — заменён на `cocktail.gif`
